1 整体规则

1.1 常用class命名定义

全局设置

大号字体:.ui-font-lg
中号字体:.ui-font-md
小号字体:.ui-font-li
夜间模式:.ui-night
无图模式:.ui-nopic

工具类

清除浮动:.ui-cf
左浮动:.ui-fl
右浮动:.ui-fr
强制显示:.ui-show
强制隐藏:.ui-hide
单行截断:.ui-text-truncate

@mixin line-clamp($lines: 2, $line-height: 1.3em) // 多行截断

      <li class="ui-border-t">上边框</li>
      <li class="ui-border">边框</li>
      <li class="ui-border-b">下边框</li>
      <li class="ui-border-tb">上下边框</li>
      <li class="ui-border-l">左边框</li>
      <li class="ui-border-r">右边框</li>
      <li class="ui-border-radius">
          圆角边框
      </li>
  </ul>

2.2 使用方式

约定缩写

缩写 原单词
-l left
-r right
-t top
-b bottom
-tb top and bottom

$ui-border: #{$namespace}border !default;

按照约定的缩写在DOM上相应位置写上对应的class名即可。

2.3 设置其他颜色

scss调用：@include border(#variable-color)

在scss文件中，相应DOM对应的选择器中，直接调用border(#variable-color)即可。

配置说明：

name type default description comments
color string #e0e0e0 颜色值 直接传入颜色值即可，也可以用色值rgb表示
举例说明：

HTML相应DOM

  ...
  <div class="ui-new">
      <div class="ui-border-tb ">红色上下边框</div>
  </div>
  ...

scss调用方式

  ...
  .ui-new {                  // 外层类名定位
    @include border(#f00); // 使用实际颜色值:`#f00`
  }
  ...

3 toast(tips)提示

3.1 使用演示

示例代码：

  <div class="ui-center">
      <div class="ui-btn" id="btn1">弹出提示</div>
  </div>

  <script type="text/javascript">
    var el,
      options = {
            content:'温馨提示内容',
            stayTime:2000,
            type:"success"
        };
    $("#btn1").tap(function(){
        el=$.tips(options);
        el.on("tips:hide",function(){
            console.log("tips hide");
        })
    });
  </script>

3.2 使用方式

  tips支持两种不同的调用方式，一般使用$.tips(options)即可。

  //最简单的方式，组件会根据默认模板输出dom结构
  $.tips(options);

  //通过传入css选择器的方式
  $("#id").tips(options);

3.3 参数设置

配置说明
name type default description
content string '操作成功' 提示内容，用来填充模板
stayTime int 1000 提示停留时间，过了这个时间自动隐藏，设置0则不自动隐藏
type string 'info' 提示类型，可选info|warn|success
eventBind string 'click' 事件绑定类型，可选click|tap
position string 'middle' 弹出提示位置，可选top|middle，top对应在顶部下拉后收起，middle对应在屏幕中间弹出后隐藏
callback funtion function(){} 回调函数

js示例：

  var el,
    tipsOption = {
      content: '操作成功', //弹出内容
      stayTime: 2000, //弹出时间 ms
      type: 'success', //弹出类型
      eventBind: 'click', //绑定类型click | tap 
      position: 'middle' // top【顶部下拉收起】 |  middle 【中间显示再隐藏】
    };
  $('#btn-tips').on(tipsOption.eventBind, function() { //该id与html中触发的dom对象相对应
    el = $.tips(tipsOption); //点击后，初始化模块
    el.on('tips:hide', function() {
      console.log('tips hide');
    })
  });

4 dialog对话框

4.1 使用演示

示例代码：

  <div class="ui-center">
      <div class="ui-btn" id="btn1">模板创建弹窗</div>
      <div class="ui-btn" id="btn2">DOM创建弹窗</div>
  </div>
  <div class="ui-dialog">
      <div class="ui-dialog-cnt">
          <div class="ui-dialog-bd">
              <div>
              <h4>标题</h4>
              <div>内容</div></div>
          </div>
          <div class="ui-dialog-ft ui-btn-group">
              <button type="button" data-role="button"  class="select" id="dialogButton<%=i%>">关闭</button> 
          </div>
      </div>        
  </div>

  <script type="text/javascript">
  $("#btn1").tap(function(){
      var dia=$.dialog({
          title:'温馨提示',
          content:'温馨提示内容',
          button:["确认","取消"]
      });

      dia.on("dialog:action",function(e){
          console.log(e.index)
      });
      dia.on("dialog:hide",function(e){
          console.log("dialog hide")
      });

  })
  $("#btn2").tap(function(){
      var dia2=$(".ui-dialog").dialog("show");
      dia2.on("dialog:action",function(e){
          console.log(e.index)
      });
  })
  </script>

4.2 使用方式

  dialog支持两种不同的调用方式，一般使用$.dialog(options)即可，不同的方式配置options后都会直接toggle弹窗。

  //最简单的方式，组件会根据默认模板输出dom结构
  $.dialog(options);

  //通过传入css选择器的方式
  $("#id").dialog(options);

4.3 参数设置

配置说明
name type default description
title string '' 浮层标题，用来填充模板
content string '' 浮层内容，用来填充模板
button array ['确认'] 浮层底部按钮的文字数组，建议不超过两个，与callback的index相互对应
select int 0 需要高亮的按钮索引（与button相互对应），高亮的按钮会添加类名select
eventBind string 'click' 事件绑定类型，可选click|tap
type number 1 弹出对话框类型，可选0|1，0指只有确认按钮操作，1则有确认和取消两个按钮操作
callback funtion function(){} 点击底部按钮后的回调函数，可以通过函数的第一个参数来获取点击的按钮索引操作（与button对应）
animation string 'pop' 弹窗弹出的动画类名，会自动为弹窗外层加上该类名
end function function(){} 弹窗弹出后或者消失后的回调，可以通过函数第一个参数来获取状态（'show'为弹出后，'hide'为消失后）

js使用示例：

  //单个按钮dialog
  var options1 = {
    title: '温馨提示', //title
    content: '温馨提示内容:anywhere you are, I am near', //the topic main content
    button: ['确认1', '取消1'], //the button name //
    eventBind: 'click', //tap | click
    type: 0  //dialog类型
  }
  $('#btn1').on(options1.eventBind, function() {
    var dia2 = $.dialog(options1); //'show'模板弹出时自动为顶级加上show类名，隐藏时自动去掉show，所以一般需要通过show类名来控制模板的显示隐藏。
    dia2.on('dialog:action', function(e) {
      console.log(e.index);
      console.log('function-- close')
    });
  })

  //两个按钮dialog
  var options2 = {
    title: '温馨提示', //title
    content: '温馨提示内容 When you need me more', //the topic main content
    button: ['确认2', '取消2'], //the button name 若type为2只能两个参数，若超过两个参数，则不展示多余参数
    eventBind: 'tap', //tap | click
    type: 1
  }
  $('#btn2').on(options2.eventBind, () => {
      console.info('dialog--->go')
      var dia = $.dialog(options2);

      dia.on('dialog:action', function(e) {
        console.log(e.index);
        console.log('function go--')
      });
      dia.on('dialog:hide', function(e) {
        console.log('dialog hide')
      });
    })
